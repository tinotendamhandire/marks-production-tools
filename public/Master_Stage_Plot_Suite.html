🎛️ Stage Plot Designer - Smart Edition

Workflow

1 · Define your crew once. Each person gets a colour.

2 · Add your acts. When you click "+ Add Act", it automatically copies everything currently on stage to save time.

3 · Drop elements on stage. Assign them to crew in the 'Selected Element' panel.

4 · Smart Changeovers: The tool automatically compares the current act to the previous act. If you add something, it calculates it as a "Bring On". If you delete something that was there previously, it strikes it and leaves a "ghost" so the crew knows where to grab it.

Views & Printing

Layout View — Clean diagram, native gear colours. Best for directors and performers.

Changeover View — Element boxes take on the assigned Crew's colour. Auto-generates aggregated "Bring on" and "Strike" lists based on differences between acts. Best for stage crew.
