🎛️ Stage Plot Designer

Workflow

1 · Define your crew once for the whole show — each person gets a colour dot.

2 · Add your acts in run-order. Drag to reorder.

3 · Place elements on the stage map. Each element has an ON/OFF state and a crew assignment.

4 · Switch between Layout view (clean diagram) and Changeover view (crew-coloured, with bring-on/strike lists).

5 · Print either view — Layout for directors, Changeover for crew.

Elements

Click a gear button to drop an element onto the stage. Drag to reposition. Double-click to rename. The dot beneath each element shows which crew member owns it.

ON / OFF state

Click an element then use the ON/OFF toggle in the element panel. OFF elements appear faded — they show where things are being struck FROM, so crew know where to go collect them.

Printing

Print Layout — clean diagram, no crew colours. For directors and performers.

Print Changeover — crew colours, bring-on/strike lists. For your stage crew.
